## Overview

SEO Head Generator is a lightweight web tool that allows developers and content creators to **quickly generate SEO-friendly `<head>` sections** for their websites. It produces properly formatted meta tags for:

- Page title and description  
- Keywords  
- Open Graph / Facebook  
- Twitter/X cards  
- Canonical URLs  

All generated code can be **copied instantly** and pasted into production.

---

## Features

- Fully dynamic form with required fields  
- Generates meta tags for SEO, Open Graph, and Twitter/X  
- Copy-to-clipboard button  
- Clean, developer-friendly design  
- Responsive and homepage-styled layout  

---

## Installation / Usage

1. Clone or download this repository.  
2. Open `index.html` in your browser to access the homepage.  
3. Click "Open Generator" or navigate to `generate.html` directly.  
4. Fill out all required fields and click **Generate**.  
5. Copy the generated `<head>` block from the output panel.
